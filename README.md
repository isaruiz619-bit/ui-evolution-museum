# ui-evolution-museum

## Project Overview

This project is a digital museum that explores the evolution of user interfaces, from early command-line systems to modern AI-assisted design. The goal is to present the history of UI as a curated exhibition, not just a website.

## Design Style

The site is inspired by **Swiss Design (International Typographic Style)**.

* clean layout and strong grid structure
* minimal color palette (black, white, neutral tones)
* clear typography hierarchy
* focus on readability and structure

This style supports the idea of presenting information clearly, like a museum exhibit.

## Cialdini Principle Used

I used the principle of **Authority**.

The site presents information in a structured, educational tone, similar to a real museum.
This builds trust and encourages users to take the content seriously and continue exploring.

## Brand Archetype

The site follows the **Sage archetype**.

This means:

* the tone is informative and educational
* the goal is to teach and explain
* the experience feels like learning from a trusted source

## Museum Experience (Curator Perspective)

If this site were reviewed by a Smithsonian curator:

Strengths:

* Clear chronological narrative (timeline of UI evolution)
* Strong visual hierarchy and readability
* Educational and informative content

Improvements:

* Could add more interactive elements
* Could improve emotional engagement (stories, user impact)
* More consistency in artifact layout


## AI Workflow (Spec-Driven Process)

I used a structured workflow:

Spec → QA → Sprint → QA → Implement → QA

Steps:

1. Planned the structure of the museum (timeline + artifacts)
2. Used AI to generate layout and content ideas
3. Reviewed and refined sections step by step
4. Fixed layout issues and image paths
5. Improved clarity and readability

## Reflection

This project helped me understand that building a good website is not just about writing code. It requires structure, planning, and iteration.

Using AI as a tool worked best when I guided it step by step instead of asking for everything at once.
